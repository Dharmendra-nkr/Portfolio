Zomato Bangalore - EDA, Geospatial Insights, and Predictions

This project performs Exploratory Data Analysis (EDA) and geospatial visualization on the Zomato Bangalore dataset to analyze restaurant distribution, popularity, and customer preferences. Using GeoPandas, we mapped high-density restaurant areas and customer hotspots. Additionally, an interactive heatmap was created and a predictive model was developed using Optuna to estimate restaurant ratings.
